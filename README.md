## Prerequisites

```
Google Chrome
./chromedriver same version as Chrome
Python 3.9+s
```

## Configuration
```
Create your own config.json and credentials.json.
Use config.json.example and credentials.json.example as your reference
```

## Install
```
pip install -r requirements.txt
```

## Run 
```
python3 telega.py
```
